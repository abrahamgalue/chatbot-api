API de Chatbot

Esta API permite enviar mensajes a un chatbot impulsado por Google Gemini 2.0 Flash.

Cómo usar la API

La API tiene un único endpoint disponible para interactuar con el chatbot.

POST /gemini

Este endpoint envía un mensaje al chatbot y recibe una respuesta basada en la conversación previa.

Parámetros requeridos (en JSON)

  • history: (opcional) Historial de mensajes anteriores para mantener la conversación.
  • message: El mensaje que deseas enviar al chatbot.

Ejemplo de solicitud

{
  "history": [
    { "role": "user", "parts": ["Hola, ¿cómo estás?"] },
    { "role": "model", "parts": ["¡Hola! Estoy aquí para ayudarte."] }
  ],
  "message": "¿Qué puedes hacer?"
}

Ejemplo de respuesta

"Puedo ayudarte a responder preguntas, crear contenido y mucho más."
